Web Apps

Random things I built for myself, with some help from Claude Code.

Task Breakdown Calendar
This is a mostly-functional calendar, with the main feature being that when you add a "task" due on some particular day, you can give the task a dozen subtasks, and the calendar will automatically distribute those subtasks to be due on days between today and the final due date. No other fancy plugins.
Vitruvian Widgets
I asked Claude Code to imagine itself as Vitruvius who was comissioned to design a website for selling widgets. I was sufficiently happy with the results that I asked Claude-Vitruvius to help me with my own website (though I still ended up doing a lot myself).
